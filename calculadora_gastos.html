Funcionalidades

Interface gráfica: Uma interface amigável permite que o usuário insira a descrição e o valor do gasto.

Validação de entrada: A calculadora valida se os campos foram preenchidos corretamente, garantindo que a descrição seja composta apenas de letras e caracteres, e o valor seja numérico.

Adicionando gastos: Ao clicar no botão "adicionar", o usuário adiciona o gasto à tabela, que é atualizada em tempo real.

Cálculo do total: A calculadora calcula e exibe o total dos gastos.

Exibição dos gastos: A tabela apresenta a descrição e o valor de cada gasto adicionado.

Detalhes da Implementação

HTML: Define a estrutura da página, incluindo os inputs para inserir a descrição e valor dos gastos, os botões, a tabela para exibir os gastos e o elemento para exibir o total.

CSS: Estiliza a aparência da página, definindo cores, fontes, espaçamento e responsividade.

JavaScript:

  • Controla a interação do usuário com a página.
  • Valida os dados inseridos pelo usuário.
  • Atualiza a tabela com os gastos adicionados.
  • Calcula e exibe o total dos gastos.

Recursos

A calculadora utiliza arrays JavaScript para armazenar os gastos inseridos.

As informações são adicionadas à tabela dinamicamente, utilizando JavaScript para manipular o DOM (Document Object Model).

O código JavaScript utiliza eventos para responder às ações do usuário.

Considerações

O projeto é simples, mas pode ser expandido com funcionalidades adicionais como:

  • Permitir a edição e remoção de gastos.
  • Adicionar categorias para organizar os gastos.
  • Salvar os gastos em um arquivo para uso posterior.
  • Criar gráficos para visualizar os gastos.
