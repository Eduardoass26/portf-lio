Calculadora de Gastos 1.0

A Calculadora de Gastos é uma ferramenta simples que permite adicionar, exibir e calcular automaticamente despesas inseridas pelo usuário, a interface valida entradas e exibe uma tabela com o total de gastos atualizado em tempo real.

Jogo da Velha

Este código implementa um jogo da velha simples em HTML, CSS e JavaScript. O jogador pode clicar nos campos do tabuleiro para marcar suas jogadas, e o código verifica se há um vencedor ou empate. Um botão "Reiniciar" permite iniciar uma nova partida.

Informações Metereológicas

Este código JavaScript usa a API OpenWeatherMap para buscar e exibir informações meteorológicas de uma cidade. O usuário digita o nome da cidade e a aplicação mostra a temperatura, descrição do clima, umidade e um ícone.

Slider

Este é um projeto de carrossel de imagens simples em HTML, CSS e JavaScript. A estrutura HTML define um carrossel que exibe uma série de imagens lado a lado, com botões de navegação para avançar ou retornar a imagem destacada.

Caderneta
