Projeto de Caderneta Infantil

Este projeto é uma aplicação desenvolvida para registrar e acompanhar o desenvolvimento do seu filho de forma personalizada e interativa, com um tema infantil de Naruto.

Estrutura do Projeto

Front-end: Utiliza React para construir uma interface amigável e intuitiva. O design infantil foi escolhido para tornar a aplicação mais envolvente e adequada ao público-alvo, com cores vibrantes e elementos gráficos que atraem e capturam o interesse infantil.

Back-end: Implementado com Node e Java, o back-end gerencia o armazenamento e manipulação dos dados inseridos na caderneta, permitindo registrar eventos, marcos, anotações e lembranças, mantendo o histórico organizado.

Funcionalidades

  • Registro de marcos e conquistas importantes no desenvolvimento do seu filho.
  • Possibilidade de adicionar observações diárias ou semanais para acompanhar mudanças e progresso.
  • Opção de adicionar fotos, imagens ou desenhos para ilustrar o crescimento e as atividades, criando uma recordação visual com o passar do tempo.

Objetivo e Usabilidade

Essa caderneta digital é uma maneira de monitorar o desenvolvimento infantil e criar uma memória detalhada da infância, acessível para revisitar no futuro.
