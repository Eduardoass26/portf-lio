Projeto de Carrossel de Imagens

Este é um projeto de carrossel de imagens simples em HTML, CSS e JavaScript. A estrutura HTML define um carrossel que exibe uma série de imagens lado a lado, com botões de navegação para avançar ou retornar a imagem destacada.

Estrutura e Funcionamento:

HTML:

  • O HTML contém três imagens (img1, img2, img3) dentro de uma div com a classe carrossel-container.
  • Cada imagem é identificada por um ID único e tem tamanho de 400x400 pixels.
  • Os botões "voltar" e "avançar" permitem navegar entre as imagens.

JavaScript:

  • Uma variável imagemAtual é usada para rastrear qual imagem está ativa.
  • A função destacarImagem ajusta a opacidade das imagens, deixando a imagem ativa com opacity: 1 e as demais com opacity: 0.3.
  • As funções avancar e retornar mudam o valor de imagemAtual para navegar para a próxima ou a imagem anterior, respectivamente. Quando o índice ultrapassa o limite, ele é redefinido para o começo ou o fim da lista.

Interatividade:

O carrossel é iniciado com a primeira imagem destacada e, ao clicar nos botões de navegação, a imagem atual é atualizada com destaque, enquanto as outras ficam esmaecidas.

Esse projeto é útil para exibir uma sequência de imagens com um efeito de foco e é fácil de adaptar para mais imagens ou novas funcionalidades.
